# Create Bemobile App

> Bemobile Typescript starter template

This is the official starter template to create Bemobile applications.

## Creating a new app

```sh
yarn create bemobile-ts-app hello-world
```

npm users

```sh
npm init bemobile-ts-app hello-world
```
